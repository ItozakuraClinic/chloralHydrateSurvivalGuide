# 写在所有的话语之前

作为人类之中的一员，我深知他们灵魂深处蜿蜒的那些于幽暗之中闪起磷光的渴望。我们渴望挣脱疼痛的枷具，渴望在平静的睡眠中与已经不在的人重逢，渴望潜入无光之海，直抵时间尽头那片诱人的黑。这一切不过是人类最为稀松平常的渴望，只是在大多时候如同星空一般伸手可及，远在天边。因着这样求而不得的痛，人类合成了诸多药物，而水合氯醛便是其中之一。笔者于2019年前后在尝试了诸多不甚完美的药物之后，想起了自己曾经在实验室中频繁用于动物实验的该药，随后在自己身上尝试。客观地说，这种药物的确在最初的一段时间为自己带来了预期之外的疗效，也正因为此，当初的自己做出了一个后患无穷的选择，即将这一药物的疗效广而告之。现如今在笔者所接触的范围内，水合氯醛已经隐隐有泛滥的趋势，而笔者自己也已经逐渐意识到这枚似乎可以把一切问题完美解决的银弹背后所隐含的代价，尚且不论该药物已经在事实上直接或间接地造成了伤害和死亡。尽管有人曾告诉我，即便自己当初没有选择将该药物的使用方法扩散，依然会有别人独立地挖掘出诸如巴比妥类药物的不安全用药方案，但大错已成，该药物的扩散趋势也已经无法挽回，对此笔者认为自己并非无辜。面对如今的现状，我们达成了这样的共识：不应当将危险药物的使用作为禁忌知识封存，而是让更多的人在彻底了解这种药物的基础上自行判断。在这本手册中，我们不会回避水合氯醛作为一种早期镇静麻醉药物的历史作用和其速效的优势，一如我们不会回避它隐藏在万能安眠药表象背后的巨大隐患。我们不希望这本手册让读者对水合氯醛产生不必要的恐惧而对它望而却步，一如我们不希望它被反向实践从而成为不顾安全的探索者手中的时间穿越指南。我们不希望这本手册所带来的讨论被限定在水合氯醛这一种药物身上，而是希望每一个读者都能够借此反思药物的局限，现代医疗和制药业的系统困境，以及我们作为血肉之躯其自身的局限。我们希望它不会成为讨论的终结，而是作为一个起点，让社群内饱受睡眠障碍和其它精神问题之苦的人们能够在充分了解的基础上，直面所有以药物为基础的治疗方案之效果和局限，我深信唯有如此，才足以挽回自己曾经所犯错误之纤毫，也唯有如此才是当前形势下，面对日后可能层出不穷的不安全用药方案最为负责的态度。
